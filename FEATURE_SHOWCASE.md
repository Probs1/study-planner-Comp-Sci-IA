# Study Planner - Feature Showcase Guide

This document provides a comprehensive walkthrough of all features in the Weekly Study Planner application for demonstration purposes.

---

## 🎯 Core Features

### 1. **Add Study Session**
- **Location**: Top-right "➕ Add Session" button or `Ctrl+N`
- **What to show**: 
  - Click the Add Session button
  - Fill in: Subject (e.g., "Mathematics"), Day (Monday), Start time (14:00), End time (16:00)
  - Choose a color from the palette or enter custom hex code
  - Add notes: "Chapter 5 - Calculus review"
  - Session appears on the calendar immediately

### 2. **Edit Existing Session**
- **Location**: Double-click any session on the calendar
- **What to show**:
  - Double-click on a session block
  - Edit dialog opens with all current details
  - Change the time, subject, color, or notes
  - Click "Save Changes"
  - Session updates in real-time

### 3. **Session Context Menu**
- **Location**: Right-click any session
- **What to show**:
  - Right-click a session block
  - Menu shows: Edit Session, Manage Tasks, Remove this block, Remove entire session
  - Demonstrate each option

### 4. **Recurring Sessions**
- **Location**: Add Session dialog
- **What to show**:
  - Click Add Session
  - Fill in session details
  - Check "Create on multiple days (recurring)"
  - Select multiple days (Mon, Wed, Fri)
  - One click creates sessions on all selected days

### 5. **Session Notes**
- **Location**: Add/Edit Session dialog
- **What to show**:
  - Create a session with detailed notes
  - Notes field supports multiple lines
  - Useful for homework details, chapters, exam prep info

---

## 📋 Task Management

### 6. **Task Lists per Session**
- **Location**: Right-click session → "📝 Manage Tasks"
- **What to show**:
  - Open task manager for a study session
  - Add tasks: "Complete homework problems 1-15", "Review lecture notes", "Practice problems"
  - Check off completed tasks
  - Delete tasks with trash icon
  - Tasks are saved automatically

---

## 📊 Analytics & Tracking

### 7. **Statistics Dashboard**
- **Location**: Top toolbar "📊 Statistics" button or `Ctrl+S`
- **What to show**:
  - Overview section: Total sessions, unique subjects, total study hours
  - Weekly goal progress bar (color-coded)
  - Time breakdown by subject
  - Demonstrates study patterns and habits

### 8. **Study Goals**
- **Location**: Tools menu → "Study Goals"
- **What to show**:
  - Set weekly study hour goal (e.g., 25 hours)
  - Goal appears in statistics dashboard
  - Progress bar updates based on scheduled sessions

---

## 🔍 Organization Features

### 9. **Filter by Subject**
- **Location**: Toolbar "📋 By Subject" button or `Ctrl+F`
- **What to show**:
  - Click filter button
  - Select a subject (e.g., "Mathematics")
  - Calendar shows only Math sessions
  - Other sessions hidden temporarily
  - Clear filter to see all sessions again

### 10. **Multi-Week View**
- **Location**: Toolbar week navigation or `Ctrl+Left/Right`
- **What to show**:
  - Click "Next ►" to see next week's schedule
  - Click "◄ Prev" to go back
  - "Today" button returns to current week
  - Plan ahead for multiple weeks

### 11. **Session Templates**
- **Location**: Tools menu → "Session Templates"
- **What to show**:
  - Save frequently used session configurations
  - Create template: "Lab Session - 3 hours - Blue"
  - Use template for quick session creation
  - Saves time when scheduling similar sessions

---

## ⚡ Smart Features

### 12. **Time Conflict Detection**
- **Location**: Automatic when adding sessions
- **What to show**:
  - Try to add a session that overlaps with existing one
  - Warning dialog appears showing conflicts
  - Option to add anyway or cancel
  - Prevents accidental double-booking

### 13. **Study Reminders**
- **Location**: Automatic background feature
- **What to show**:
  - Sessions trigger reminders at:
    - 60 minutes before start
    - 30 minutes before start
    - At session start time
  - Popup notifications alert you

### 14. **Break Reminders**
- **Location**: Tools menu → "Break Reminder Settings"
- **What to show**:
  - Configure break intervals (e.g., every 50 minutes)
  - Set break duration (e.g., 10 minutes)
  - Helps maintain healthy study habits

---

## 🎨 Customization

### 15. **Dark Mode**
- **Location**: View menu → "Toggle Dark Mode" or `Ctrl+D`
- **What to show**:
  - Toggle between light and dark themes
  - Easier on eyes for night studying
  - Note: Currently shows message to restart app

### 16. **Custom Colors**
- **Location**: Add/Edit Session dialog
- **What to show**:
  - Quick color palette for common choices
  - Custom hex code entry for any color
  - Live preview of selected color
  - Color-code subjects (Blue=Math, Red=English, etc.)

---

## 📅 Visual Indicators

### 17. **Today Indicator**
- **Location**: Automatic on calendar
- **What to show**:
  - Current day column highlighted in red
  - Shows ⭐ star next to day name
  - Easy to spot current day at a glance

### 18. **Current Time Line**
- **Location**: Automatic on today's column
- **What to show**:
  - Red line shows current time
  - Updates every minute
  - Helps see what's happening now vs. upcoming

---

## 💾 Data Management

### 19. **Export Sessions**
- **Location**: File menu → "Export as JSON" or `Ctrl+E`
- **What to show**:
  - Export all sessions to JSON file
  - Backup your schedule
  - Share with others or move to another device

### 20. **Import Sessions**
- **Location**: File menu → "Import from JSON"
- **What to show**:
  - Import previously exported sessions
  - Combine schedules from different sources
  - Automatically generates unique IDs

### 21. **Session History/Archive**
- **Location**: Tools menu → "Archive/History"
- **What to show**:
  - View all sessions in table format
  - Sort by subject, day, or time
  - Shows duration and notes
  - Export history to JSON

---

## ⌨️ Keyboard Shortcuts

### 22. **Quick Access Shortcuts**
- **Location**: Help menu → "Keyboard Shortcuts"
- **What to show**:
  - `Ctrl+N` - New session
  - `Ctrl+E` - Export
  - `Ctrl+S` - Statistics
  - `Ctrl+F` - Filter
  - `Ctrl+D` - Dark mode
  - `Ctrl+Left/Right` - Week navigation
  - `Delete` - Delete session info
  - `Double-click` - Edit session
  - `Right-click` - Context menu

## 💡 Tips for Best Demo Video

1. **Use sample data**: Create diverse sessions across different subjects, days, and times
2. **Color code subjects**: Math=Blue, English=Red, Science=Green, History=Yellow
3. **Show real scenarios**: "Exam prep week", "Regular study schedule", "Project deadlines"
4. **Highlight smooth workflow**: Show how quickly you can add/edit sessions
5. **Demonstrate problem-solving**: Show conflict detection catching double-booking
6. **Use realistic notes**: "Review chapters 1-3", "Complete assignment due Friday"
7. **Show completed tasks**: Demonstrate checking off tasks as done
8. **End with statistics**: Show the dashboard summarizing all scheduled study time
